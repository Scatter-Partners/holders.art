# holders-art
Holders.art is a multi-snapshot tool.

https://holders.art/

## Development

You can open the `index.html` file directly in a browser.

Or run a lightweight local web server:

```bash
docker run -it --rm -p 3000:80 -v "$PWD":/usr/local/apache2/htdocs:ro httpd:2.4-alpine
# then navigate to http://localhost:3000
```
